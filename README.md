## PizzaStore-MinimalAPI


A simple ASP.NET Core Minimal API for managing pizza data, built as part of Microsoft Learn.

## Overview

This project demonstrates how to build a RESTful API using:
- ASP.NET Core Minimal APIs
- Entity Framework Core
- SQLite (or your configured database)

It supports basic CRUD operations for managing pizzas.

---

## Features

- Get all pizzas
- Get pizza by ID
- Create a new pizza
- Update an existing pizza
- Delete a pizza
- Database integration using Entity Framework Core

---

## Technologies Used

- C#
- ASP.NET Core Minimal API
- Entity Framework Core
- SQLite
